Anatomy of CSS Rules

The anatomy of a CSS rule consists of a selector and a declaration block. The selector identifies which HTML element(s) the rule will style, such as a tag (p), class (.container), or ID (#header). The declaration block is enclosed in curly braces { } and contains one or more propertyâ€“value pairs that define how the selected elements should appear. Each declaration includes a property (like color, font-size, or margin) followed by a colon and a value that specifies the setting for that property, ending with a semicolon. Together, the selector and declarations form a complete CSS rule that controls the presentation of elements on a webpage.
